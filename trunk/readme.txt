This BuddyPress media component adds features like photos, videos and audio upload to BuddyPress with activity streaming.
This version is for BuddyPress 1.2.5.x

New Features Added are:

* Media menu is added to BuddyPress user profile pages
* Rating/Ranking of Photo / Video
* Album support to organize photos
* Sidebar widgets to display media anywhere in your theme
* Report abuse button
* View Count
* Ajax-based interface for inline editing of media title/description

Upcoming Features are:

* HTML5 player/uploader support for iPhone and other non-flash devices
* Managing User Upload quota
* Granular privacy control over uploaded media
* Tagging for media-files
* Custom page-templates for different media types
* Localization/Translation for different languages
* Group uploading feature to upload media to specific groups
* Privacy controls to decide who can see your media files
* Photo-tagging support for photos uploaded using this plugin

== Description ==
This BuddyPress media component adds features like photos, videos and audio upload to BuddyPress. These features are essential for building a social network around BuddyPress.

= Features =

* Media menu is added to BuddyPress user profile pages
* Rating/Ranking of Photo / Video
* Album support to organize photos
* Sidebar widgets to display media anywhere in your theme
* Report abuse button
* View Count
* Ajax-based interface for inline editing of media title/description
* BuddyPress user gets a media menu on their profile pages
* The users can upload video, images and audio files seamlessly to their profiles. (Data will be stored on the Kaltura server by default. You can use your own server as an alternative.)
* Buddypress users don't need a separate Kaltura account to use media-rich features added by this component.
* Only Buddypress (Wordpress MU) admin needs a Kaltura Partner ID once. You can get it free from kaltura.com

== Installation ==

1. Download component source code and unzip it.
2. Upload "buddypress-kaltura-media-component" directory into WordPress plugins directory i.e. "wp-content/plugins".
4. Go to "Dashboard >> Plugins" menu and activate "Buddypress-Kaltura Media Component".
5. Go to "Dashboard >> Settings >> Kaltura Settings" and enter your Kaltura Partner ID there. You can get <a href="http://corp.kaltura.com/about/signup">Kaltura Partner ID for free</a>.


That's it ... Have fun!

== Frequently Asked Questions ==
<strong>Q. Where can I get Partner ID.</strong>

Go to http://corp.kaltura.com/about/signup and you will get Partner ID for free.
Same PartnerID, emailID and password should be used to complete installation of this plugin.
Throughout the installation, only this PartnerID is required. No need to get PartnerID for every WPMU user.

<strong>Q. Where to Insert PartnerID to make this component work</strong>

After 'Activate'ing Component/Plugin, click on setting link. Submit PartnerID, EmailID and Password, and Done!
